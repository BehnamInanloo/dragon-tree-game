game rules

  • Each player must get dice #6 to start the game.
  • From this point on, getting the dice #6 will reward each player with a chance to re-roll the dice.
  • In this game, it is possible to hit another player. In this case, the other player will return to the beginning of the game.
  • If a player enters the dragon's house, he will be damaged and randomly return to one of the previous cells.
  • If a player enters a tree house, he can climb the tree and randomly jump to one of the next cells.
  • Each player who crosses the final flag first will win the game.
  • You can change the placement of dragons and trees using the reset button.
